# \#🎯 Objetivo do Software

# Desenvolver um sistema desktop para gerenciar o cadastro de funcionários, clientes, motoristas e veículos de uma transportadora, garantindo a integridade e persistência dos dados.

# 

# \#📝 Funcionalidades do Sistema (Requisitos)



# \-Login e Autenticação de usuários.



# \-Cadastro de Funcionários: CRUD (Criar, Ler, Atualizar, Deletar).



# \-Cadastro de Clientes: CRUD.



# \-Cadastro de Motoristas: CRUD.



# \-Cadastro de Veículos: CRUD.



# \-Conexão e persistência de dados utilizando MySQL.-PI
